  • 30 Mar. 2018

  • New homepage

    For upcoming third alpha version I did some homepage. You are viewing it ;-)

  • Major refactoring and luaunit selftests

    Version 0.0.3 will come with major refactoring in class manager. Some nice stuff (mixins, friend classes, annotations). See the docs for details. By using luaunit we will be able to create test reports for xwos and -later on - applications.

  • Jenkins and github setup

    It took some time to setup a jenkins build as well as a github mirror. Now every upcoming version will be tested automatically. The test report can be viewed as web site as well as FLV video.
  • 25 Mar. 2018

  • ComputerCraft - Unicode support

    To get some nice drawings in GUI I decided to add unicode compatibility to ComputerCraft. It took some days and some discussion in community. See the first, second and last but not least the third try at github. Hope that dan merges this pull request one day.
  • 14 Mar. 2018

  • Published v0.0.2

    Version 0.0.2 contains a major refactoring with some class loader and it contains libraries allen/moses. Now it was time to experiment with CCLite, a ComputerCraft emulator. It took some time to add compatible code.
  • 4 Mar. 2018

  • Published first alpha

    The day everything started. Not really. I already worked on XWOS some days before. But now it was time to create a forum post at ComputerCraft forum. The first version does some self test but under the hood there are some interesting features.
