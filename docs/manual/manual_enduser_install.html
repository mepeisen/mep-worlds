Resource pack

Installation via resource pack is recommended if You have control over your server. Notice that this resource pack must be installed on servers or single player games. If you install it in clients and then remotely connect to a minecraft server it won't work for you.

Download xwos.zip and put it to your minecraft resource packs folder. You will find it within your minecraft installation. It is simply called "resourcepacks". You do not need to extract it.

Pastebin installer

There is a pastebin installer. By using this installer you can setup XWOS on every server having installed ComputerCraft but not having the resource pack. There are three differences You should always remember:

  • Installing via pastebin must be done on every single device that will use XWOS.
  • You or other users may be able to hack into the XWOS files because they are writable.
  • It requires disk space because XWOS is saved as regular files onto your device.

So if you need additional security on multi user servers You should choose the resource pack to install XWOS into read only memory.

To install XWOS simply type following command into your console: 

pastebin run cst2hmTN
