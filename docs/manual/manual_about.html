About XWOS

XWOS is a ComputerCraft operating system for ComputerCraft devices. It provides a modern system for applications. You can install XWOS either via resource pack (recommended) or via lua installer. However for getting more security You should prefer using the resource packs because they are using the ROM and cannot be "hacked".

Once installed XWOS can be used by any minecraft player having access to your ComputerCraft device. Like modern computer operating systems XWOS features multiple users and processes strictly isolated from each other. That means: You can lock the desktop and other users may login and start their own apps not influencing your work.

For more detailed explanation see the features.
