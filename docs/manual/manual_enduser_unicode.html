About Unicode

ComputerCraft uses some small character set: ISO-8859-1. It contains ASCII characters and some more things. But it does not support many languages or graphical symbols. For best user experience I recommend to install full unicode support. Unicode supports (theoretically) over one million different characters. For more details You can read WikiPedia. XWOS itself will work on non unicode ComputerCraft.

Activate Unicode

First of all You need a custom build. Dan did not yet merge the Unicode support into official build. View my comment at GitHub. You will find the custom plugin jar. Replace this jar with your current version in mods folder of your plugin installation. If you do not trust the JAR you can review my changes and build it from source.

Second task is to install a unicode font that covers all of the known characters. I decided to use GNU Unifont. You can find the corresponding resource pack within the same link. Download it and copy it to the resourcepacks folder of your minecraft installation. Why do you need an additional resource pack? The main reason is that GNU Unifont is published under terms of GPL and this is incompatible to ComputerCrafts license. So we cannot ship ComputerCraft together with Gnu Unifont.

Important: After starting your minecraft you need to activate the resource pack. If you do not do it you will get a broken display as soon as you switch the font.

Using GNU Unifont

XWOS will automatically assume that GNU Unifont is available as soon as it detects the unicode support. If some client does not know how to handle GNU Unifont he will still use the old font and see "?" characters instead of nice unicode characters. So please inform Your friends to install GNU Unifont too.

If you like to use GNU Unifont outside you can activate it by following code line (see sample code in my GitHub comment): 

term.native().setFontName("unifont")
