Overview

Allen is a LUA library for string handling. The source code can be browsed at Github. You can read the official documentation. XWOS supports version 1.1 of allen. The API reference seems to be dead. But you can read the comments directly in source code starting with line 154.

The functions are loaded into string API. For example the startsWith function is available via string.startsWith. The following features are NOT (yet) supported because of limitations in LUA version or in ComputerCraft itself:

  • built-in string library
  • string indexing
  • strings as functions

If your ComputerCraft supports utf8 there will be a utf8 variant of allen loaded into utf8 API. So all functions of allen will be available there. For details on utf8 support see unicode enduser documentation.
